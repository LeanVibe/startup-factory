# Project Brief: Startup Factory

## Problem
- **Who is the user?** Non-technical founders, entrepreneurs
- **What pain is urgent + expensive?** Need to validate business idea with MVP but lack technical skills

## Promise (Painkiller)
- In 1 sentence: Transform business idea into live MVP in 25 minutes through intelligent AI conversation
- What is the narrow wedge? 15-minute AI interview → business logic generation → code generation → deployment

## ICP
- **Role:** Non-technical founders, entrepreneurs
- **Context:** Have business idea but can't code
- **Budget:** Founder tool pricing ($99-299/month)
- **Trigger:** Need MVP to validate idea quickly

## MVP Scope (14 days)
- **Critical path:**
  - AI founder interview system
  - Business blueprint generator
  - Code generator
  - Deployment automation
- **Out of scope:**
  - Advanced customization
  - Multi-tenant support

## Pricing Hypothesis
- **Price:** $99/month for individuals, $299/month for teams
- **Packaging:** Individual: 1 MVP/month, Team: unlimited MVPs

## Risks / Assumptions
- Top 3 assumptions to validate first:
  1. Founders will pay $99/month for MVP generation
  2. 25-minute promise is achievable
  3. Generated MVPs are production-ready enough

## Status
- **Intake Date:** 2026-01-20
- **Stack:** Python, FastAPI, PostgreSQL, Redis, Lit Web Components
- **Repository:** Active git repository
- **Features:** AI interview system, business blueprint generator, smart code generator, deployment automation, industry compliance (HIPAA, PCI, FERPA)
